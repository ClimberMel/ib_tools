Welcome to IB_TOOLS documentation!
===================================

**IB_TOOLS** is a Python library for *ib_insync* samples.
ib_insync 'https://github.com/erdewit/ib_insync' offers a *simple* and *intuitive* API to Interactive Brokers.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
